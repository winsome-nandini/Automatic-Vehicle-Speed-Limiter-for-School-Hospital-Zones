# Automatic Vehicle Speed Limiter

### Smart Zone-Based Vehicle Speed Control System for School and Hospital Zones

An intelligent transportation safety system designed to automatically regulate vehicle speed in restricted zones using RFID-based detection, real-time speed monitoring, and embedded control to reduce overspeeding and improve road safety.

---

## About The Project

Automatic Vehicle Speed Limiter is designed to address the growing concern of overspeeding in safety-critical areas such as schools, hospitals, residential communities, and pedestrian zones.

Traditional speed control methods such as warning signs, speed breakers, and manual traffic enforcement rely heavily on driver awareness and compliance. However, these methods are often insufficient to prevent overspeeding and accidents.

This project provides an automated speed-governance solution by combining intelligent zone detection, embedded control, and real-time speed regulation.

The system consists of:

* RFID-Based Zone Detection System
* Vehicle Speed Monitoring and Control Unit

The system automatically detects restricted zones, activates speed-limiting mode, and ensures the vehicle does not exceed the configured speed threshold.

---

## Why Automatic Vehicle Speed Limiter?

Overspeeding near schools and hospitals can lead to:

* Road accidents
* Pedestrian injuries
* Reduced public safety
* Traffic violations
* Life-threatening collisions

Current safety measures mostly depend on human behavior and enforcement.

Automatic Vehicle Speed Limiter addresses these challenges by combining:

* Intelligent zone detection
* Automatic speed enforcement
* Real-time speed monitoring
* Wireless connectivity
* Smart transportation integration

This creates a proactive safety system capable of preventing overspeeding before accidents occur.
